Tag-Do

solo project

Tired of having a stupidly long to-do list, me too.

Tag-Do is my custom do to list website that allows you to set both the priority and a unique tag to any entry, allowing you to go and search for and sort for your task by tags!

Currently still in development, and is setup for a single user, I plan to make the system feature a login where unique users can have their own lists.

Check it out

Unitec Pathway Planner

Group project

Unitec has a notoriously difficult process for finding class information, my group proposed a solution where in which we would create a website for students to use that would show all relevant information regarding career pathways in a single, easy to navigate location.

As with all gorup projects, this didn't work out as planned. But the software as it stands is viewable with our test data in an unfinished state.

I wrote the PHP for the pages to handle data being displayed.
